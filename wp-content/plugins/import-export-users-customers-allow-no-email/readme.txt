== Installation ==

### **Installation**

*   Install **Import and export users and customers - Allow No Email addon** by uploading the ZIP file in the Add new plugin form into WordPress dashboard.
*   Then, after the package is uploaded and extracted, click&nbsp;_Activate Plugin_.
*   Activate the license you will get by email

If you get any error after following through the steps above please contact us through email in contact@import-wp.com so we can get back to you with possible helps in installing the plugin and more.
