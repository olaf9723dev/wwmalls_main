== Description ==
Local Pickup Pro for WooCommerce plugin is shipping method for WooCommerce allows your customers to come to you to pick up their purchased products at store location.
### Features And Options:
* Easily add stores in admin panel.
* Easily cuustomize stores with include and exclude in backend.
* Dropdown of stores in Checkout page.
* Add custom cost or discount per store.
* Add Pickup store details in order.
== Installation ==
= Requires WooCommerce =
1. Upload the plugin files to the '/wp-content/plugins/local-pickup-pro-for-woocommerce' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to settings page from `Admin menu > Local Pickup Pro > Settings` or the shipping methods page in Woocommerce to activate `Local Pickup Pro Shipping` shipping method.
4. Done.
== Frequently Asked Questions ==
= How to setup? =
Just activate the plugin, go to settings page and enable the shipping method. Customize the shipping method title.
